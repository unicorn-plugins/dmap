# PPT 작성 가이드 

## 이미지 작성
- 특정 부분의 내용이 PPT 도형보다 이미지로 작성하는 것이 더 좋다고 판단되는 경우 이미지로 생성  
- `{PLUGIN_DIR}/resources/resource.md'에서 이미지 생성 프로그램을 찾아 작성 

## PPT 스타일시트:  
```

## 5. 디자인 규칙

### 필수 준수 사항

- ✅ 흰색 배경(`#FFFFFF`) 위에 어두운 텍스트 사용 — 최소 대비율 4.5:1 준수
- ✅ 타이틀 바는 항상 슬라이드 최상단 전체 너비(`1152pt`) 차지
- ✅ 모든 콘텐츠는 좌우 40pt 여백 안에 배치
- ✅ 본문 텍스트 최소 크기 14pt (가독성 보장)
- ✅ 색상 스와치에는 반드시 HEX 코드 라벨 포함

### 주의 사항

- ⚠️ 마스터 배경(Teal)이 투과될 수 있으므로 커스텀 슬라이드에는 반드시 흰색 배경 사각형 배치
- ⚠️ 서로 다른 슬라이드 마스터 간 레이아웃 ID를 교차 참조하지 않을 것
- ⚠️ 푸터 영역에는 마스터 요소가 있으므로 콘텐츠 배치 금지
- ⚠️ `#FFFFFF` 스와치에는 `#CBD5E1` 테두리를 추가하여 시인성 확보

```
